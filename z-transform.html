🔄 Z-Transform

The Z-Transform is a fundamental tool in the analysis and design of discrete-time signals and systems. It is the discrete counterpart to the Laplace Transform.

📄 Definition

The Z-Transform of a discrete signal x[n] is defined as:

X(z) = ∑ x[n] z−n

Where the summation is taken over all integer values of n for which the sum converges.

🧩 Common Z-Transform Pairs

  • δ[n] → 1
  • u[n] → 1 / (1 − z−1), |z| > 1
  • aⁿ u[n] → 1 / (1 − a z−1), |z| > |a|
  • n aⁿ u[n] → a z−1 / (1 − a z−1)²

📘 Properties

  • Linearity: Z{a·x[n] + b·y[n]} = a·X(z) + b·Y(z)
  • Time Shifting: Z{x[n − k]} = z−k X(z)
  • Scaling: Z{aⁿ x[n]} = X(z / a)
  • Convolution: Z{x[n] * h[n]} = X(z)·H(z)

🎯 Region of Convergence (ROC)

ROC: Set of z values for which the Z-transform converges. Crucial for stability and causality.

📈 Pole-Zero Plot

System is stable if all poles lie inside the unit circle.

📚 Example

Example: Find Z{(1/2)n u[n]}

Using table: X(z) = 1 / (1 − 0.5 z−1), ROC: |z| > 0.5

🧠 Tips

  • Always state ROC in your final answer
  • Use long division or partial fractions to simplify inverse Z-transforms
  • Compare with Laplace for intuition: z = esT

📝 Quiz

Q1: Z{δ[n − 2]}

Answer: z−2

Q2: Z{n (0.3)n u[n]}

Answer: (0.3 z−1) / (1 − 0.3 z−1)²
