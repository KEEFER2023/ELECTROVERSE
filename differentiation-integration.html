Differentiation & Integration

This page offers a complete, in-depth guide on differentiation and integration for electrical engineering students — packed with theory, detailed examples, EE relevance.

🔹 Section 1: Differentiation

Definition: Differentiation measures the instantaneous rate of change of one quantity with respect to another (e.g., voltage change over time).

EE Insight: Di/dt determines the voltage across an inductor: vL(t) = L·[di/dt].

Fundamental Rules

  • Power rule: d/dt[tⁿ] = n·tⁿ⁻¹
  • Constant multiple: d/dt[k·f(t)] = k·f′(t)
  • Sum rule: d/dt[f+g] = f′ + g′
  • Product rule: d/dt[f·g] = f′g + fg′
  • Chain rule: d/dt[f(g(t))] = f′(g(t))·g′(t)

Detailed Worked Examples

Example 1 (Power Rule):
f(t) = 7t⁵
Step-by-step:
- Apply power rule: d/dt[t⁵] = 5t⁴
- Multiply constant: 7 × 5t⁴ = 35t⁴
Example 2 (Trigonometric):
f(t) = 4 sin(3t)
- Derivative of sin(3t) is cos(3t)×3
- f′(t) = 4 × 3 cos(3t) = 12 cos(3t)
Example 3 (Product Rule):
f(t) = t²·e3t
Apply product rule:
- f′ = 2t·e3t + t²·3e3t
- Combine: e3t(2t + 3t²)
Example 4 (Chain Rule):
f(t) = cos(5t²)
- Inner: 5t² → derivative =10t
- f′ = –sin(5t²) × 10t = –10t sin(5t²)

Study Tip:

When mixing rules (product + chain), annotate each part (e.g., outer & inner functions), then apply the rule step-by-step.

🔸 Section 2: Integration

Definition: Integration finds the accumulated value — like the total charge from a current over time or area under a signal curve.

EE Insight: Voltage across a capacitor: vC(t) = (1/C) ∫ i(t) dt.

Key Integration Techniques

  • Power rule: ∫ tⁿ dt = tⁿ⁺¹/(n+1) + C
  • Constant multiple: ∫ k·f(t) dt = k ∫ f(t) dt
  • Sum rule: Integral of f+g = ∫f + ∫g
  • Substitution: Use when integrand looks like f(g(t))·g′(t)
  • Integration by parts:
    ∫ u dv = u v – ∫ v du

Detailed Worked Examples

Example 5 (Power Rule):
∫ 10t³ dt
- Integral: (10/4) t⁴ = 2.5t⁴ + C
Example 6 (Trigonometric):
∫ sin(2t) dt
- Anti-derivative of sin(ax) = –cos(ax)/a → –(1/2)cos(2t) + C
Example 7 (Substitution):
∫ t·et² dt
- u = t² → du = 2t dt
- Integral = (1/2) ∫ eu du = (1/2)et² + C
Example 8 (Integration by Parts):
∫ t ln(t) dt
- Let u = ln(t), dv = t dt → du = dt/t, v = t²/2
- Result: (t²/2)*ln(t) – (1/2)*(t²/2) + C = (t²/2)[ln(t) – 1/2] + C

Study Tip:

Label u, dv clearly when using parts. For substitutions, always replace both dt and remaining t variables.

🧩 Section 3: Interactive Quiz & Worked Solutions

Quiz 1: d/dt [3t⁴ – 5t + 7] = ?

Solution:
- Derivative of 3t⁴ = 12t³
- Derivative of –5t = –5
- Constant → 0
Answer: 12t³ – 5

Quiz 2: d/dt [e–t·t²] = ?

Solution:
Use product rule:
f = t², g = e–t
f′ = 2t, g′ = –e–t
Result: 2t·e–t + t²·(–e–t) = e–t(2t – t²)

Quiz 3: ∫ (6t + 4) dt = ?

Solution:
- ∫6t dt = 3t²
- ∫4 dt = 4t
Answer: 3t² + 4t + C

Quiz 4: ∫ t·e3t dt = ?

Solution:
Use integration by parts:
u = t → du = dt
dv = e3t dt → v = e3t/3
Result: (t·e3t/3) – ∫ (e3t/3) dt = t·e3t/3 – e3t/9 + C = e3t(t/3 – 1/9) + C

Overall Tip: Show all interim steps in exams — annotate which rule you're using (e.g., power, product, chain, substitution, parts). That clarity earns marks and understanding.
