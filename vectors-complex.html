Vectors and Complex Numbers

Master directional quantities (vectors) and oscillatory representations (complex numbers & phasors) — essential for fields, circuits, power, and signal analysis in EE.

1. Vectors in EE

Vectors combine magnitude + direction. Key in representing fields, forces, and multi-phase currents.

Diagram: Vector A, its x and y components.

Vectors are used to calculate electric fields, forces, and align currents — critical in real-life circuit and field problems.

Vector Formulas

  • |A| = Ax i + Ay j (+ Az k)
  • |A| = √(Ax² + Ay²), angle θ = atan2(Ay, Ax)
  • Dot product: A·B = AxBx + AyBy = |A||B|cosθ
  • Cross product magnitude: |A×B| = |A||B|sinθ, direction via right-hand rule
Example:
A = 3i + 4j, B = 4i – 2j
|A|=5, |B|=√(16+4)=√20≈4.47
Dot: 3×4 + 4×(–2) = 12 – 8 = 4 → cosθ = 4 / (5×4.47) ≈ 0.179 → θ ≈ 79.68°
Cross (2D scalar): A×B = (3×–2) – (4×4) = –6 – 16 = –22 → magnitude 22 (into page)

2. Complex Numbers and Phasors

Complex numbers allow representation of oscillatory signals using magnitude + phase.

Argand plane: point z = a + jb as vector from origin.

Forms and Operations

  • Rectangular: z = a + jb
  • Polar: z = r∠θ, r = √(a²+b²), θ = atan2(b,a)
  • Euler: z = rejθ
  • Add/Subtract: component-wise
  • Multiply/Divide: polar form — multiply magnitudes, add/subtract angles
Example:
z₁ = 3+4j → r=5, θ≈53.13° → 5∠53.13°
z₂ = 1–j → r=√2, θ=–45° → multiply: 5√2∠8.13°
Divide: (5/√2)∠(53.13+45)=3.536∠98.13°

Solution:

Q4: Multiply z₁ = 4∠30° and z₂ = 2∠–45°

Solution:
Multiply magnitudes: 8
Add angles: 30 + (–45) = –15°
Result: 8∠–15°

Solution:
Sum = (3+1) + j(4–2) = 4 + 2j → polar: r=√20≈4.472, θ=26.565°

4. EE Applications

  • Phasors: Represent AC voltages/currents as complex numbers (V=Vm∠φ)
  • Impedance: Z = R + jX — addition for series, reciprocal for parallel
  • Power: P = VIcosφ — dot product in phasor form
Example:
R=10Ω, L=0.1H, f=50Hz → ω=314
ZL=jωL=j31.4, Ztotal=10+j31.4 = 10∠health-check ~10∠72.6°

5. Summary and Tips

  • Draw vectors, decompose early in problems
  • In AC use polar for multiplication, rectangular to add
  • Convert between forms quickly using practice
  • In exams, write every step: magnitude, angle, form
Exam Tip: Use atan2(y, x) to get full-angle accuracy including quadrants.
