💾 Digital Electronics

Digital electronics deals with circuits that operate using discrete signals — typically binary — to represent logical and numerical values. It’s the foundation of computers, microcontrollers, digital communication, and control systems.

🔣 Boolean Algebra

Boolean algebra is a mathematical framework for expressing and analyzing logic operations in digital circuits. Named after George Boole, it forms the basis of logic gates and digital design.

📗 Boolean Variables & Operations

  • Variables: Take only two values → 0 (false) and 1 (true)
  • Basic Operations:
    • AND (·): Output is 1 only if both inputs are 1
    • OR (+): Output is 1 if either input is 1
    • NOT (‾): Inverts input (0 becomes 1, 1 becomes 0)
Example: A = 1, B = 0
A · B = 0
A + B = 1
¬A = 0

🧾 Boolean Laws & Theorems

  • Identity Law: A + 0 = A, A · 1 = A
  • Null Law: A + 1 = 1, A · 0 = 0
  • Idempotent Law: A + A = A, A · A = A
  • Involution: ¬(¬A) = A
  • Complement Law: A + ¬A = 1, A · ¬A = 0
  • Distributive: A · (B + C) = AB + AC
  • De Morgan’s: ¬(A · B) = ¬A + ¬B; ¬(A + B) = ¬A · ¬B
💡 Use De Morgan’s Theorems to convert expressions involving NAND/NOR gates into simpler AND/OR/NOT form.

🧮 Boolean Simplification

Logic minimization reduces cost and complexity of digital circuits, making them faster and more efficient. Use laws or Karnaugh Maps for simplification.

Example: Simplify F = A·¬B + A·B
Step 1: Factor A ⇒ A(¬B + B)
Step 2: Use Complement Law ⇒ A(1) = A
